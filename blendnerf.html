3D-aware Blending with Generative NeRFs

Hyunsu Kim1, Gayoung Lee1, Yunjey Choi1, Jin-Hwa Kim1,2, Jun-Yan Zhu3
1NAVER AI Lab, 2SNU AIIS, 3CMU
Paper (High res) arXiv Code

BlendNeRF can align and compose unaligned images without manual effort.

Abstract

Image blending aims to combine multiple images seamlessly. It remains challenging as input images are often captured by different cameras under various lighting conditions. Existing 2D-based methods struggle when input images are misaligned due to differences in 3D camera poses and object shapes.

To address the above issues, we propose a 3D-aware blending method using generative Neural Radiance Fields (NeRF), including two key components: 3D-aware alignment and 3D-aware blending. For 3D-aware alignment, we align the reference image globally by changing the camera pose of the generative NeRF and then locally align each part using a 3D transformation matrix. To further leverage 3D information of objects, we propose 3D-aware blending that directly blends images on the generative NeRF's latent representation space, rather than raw pixel space.

Collectively, our 3D-aware blending method outperforms existing 2D baselines, as validated by extensive quantitative and qualitative evaluations with FFHQ and AFHQ-Cat.

Comparison with Baselines

Comparison with the existing blending methods. (a) Baseline. 2D blending methods compose two images without any 3D-aware alignment. (b) Baseline with 3D-aware alignment. To address misalignment, we apply our 3D-aware alignment method to existing 2D blending methods. (c) Proposed method. We propose 3D-aware blending after applying our 3D-aware alignment. Note that all methods do not use any 3D labels or 3D morphable models.

3D-aware Alignment

Global

Global alignment is an essential part of the alignment. Rotation alone can make huge changes in the image space. In contrast to 2D GANs, generative NeRFs G can resolve this issue by novel-view synthesis.

Local

Local alignment is a fine-grained alignment between the target regions of the images. Even though we match the rotation of images, the scale and translation of target regions (e.g., face, eyes, etc.) can be different.

3D-aware Blending

Our 3D-aware blending pipeline. We aim to find the best latent code wedit that generates a well-blended image. To achieve this goal, we exploit both 2D pixel constraints (RGB value) and 3D supervision (volume density). Using the image-blending and density-blending losses, we optimize the latent code wedit.
